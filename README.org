* Portfolio Tracker
  A simple program to track your portfolio and for doing papper trading.
  
  This is my attempt to try and create a servive that uses an API. 
  As always feel free to provide feedback. 

** Dependencies
  - Junit
  - Maven
  - Spring Framework
  - javax.servlet 
  - Tomcat 8
  - OkHttp3
  - Jackson
